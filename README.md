# myLibrary
A Vue.js / Express.js web application to manage your home book library

# Setup

You need to have Node Js installed: https://nodejs.org/en/

I suggest you install:

VSCode: https://code.visualstudio.com/


Git: https://git-scm.com/

### Client - Terminal A
```
cd client
npm install
npm start
```

### Server - Terminal B
```
cd server
npm install
npm start
```
